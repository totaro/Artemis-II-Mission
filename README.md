# 🚀 Artemis II Mission Tracker & Archive

Welcome to the **Artemis II Mission Tracker**, a dedicated web application built to document and celebrate humanity's return to deep space. 

In April 2026, NASA's Artemis II mission sent four astronauts on a historic lunar flyby. This project serves as a living archive of that journey, featuring a curated, high-resolution gallery of the most stunning photographs taken by the crew—from breathtaking Earthsets over the lunar horizon to the first human views of the lunar far side in over 50 years.

### ✨ Features
* **Mission Timeline:** Follow the key milestones of the Artemis II journey.
* **Curated Visual Archive:** A stunning, alternating-layout gallery of high-resolution images sourced directly from the official NASA Image and Video Library.
* **Detailed Metadata:** Each capture includes official NASA descriptions, dates, and location data (e.g., Orientale Basin, South Pole-Aitken basin).
* **Responsive Design:** Built to look beautiful on both desktop and mobile devices.

### 🛠️ Built With
